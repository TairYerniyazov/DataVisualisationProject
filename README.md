This repository contains the code and data for a data visualization project on EU data analysis. The project involves creating visualisations of a variety of EU datasets, including:

* adjusted salary;
* CO2 emissions;
* life expectancy;
* unemployment rates;
* infographics.

> The code for the visualisations is written in Python using the Jupyter Notebook (tested in Google Colab). Each of the folders mentioned above (except the one dedicated to infographics) contains a Colab notebook.

**ðŸ’° Adjusted salary:** the notebook visualises the average salary in each EU member state. This allows us to compare salaries across different countries.

**ðŸ§“ðŸ» Life expectancy:** the notebook visualises the average life expectancy in each EU member state. You can easily compare it as we prepared some maps for it as well.

**ðŸ’¼ Unemployment rates:** the notebook visualises the unemployment rate in each EU member state. This allows us to track changes in the unemployment rate and to compare it across different countries.
